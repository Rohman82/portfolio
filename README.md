# 👤 Personal Portfolio — Rohman

Website portofolio pribadi sebagai Frontend Developer, menampilkan project, sertifikat, dan informasi kontak dalam satu halaman.

**🔗 Live Site:** [portfolio-rohman.vercel.app](https://portfolio-rohman.vercel.app)

## ✨ Fitur

- Hero section dengan efek mengetik otomatis dan avatar bergaya oval-glow
- Showcase 3 project dengan link live demo dan GitHub
- Section sertifikat dengan link verifikasi
- About section dengan daftar skill
- Contact section (email & WhatsApp langsung terhubung)
- Navigasi aktif otomatis mengikuti section yang sedang dilihat
- Full responsif di semua ukuran layar

## 🛠️ Tech Stack

- React
- Tailwind CSS v4
- Vite
- Font Awesome (ikon)
- Google Fonts (Poppins)

## 🚀 Menjalankan Secara Lokal

```bash
git clone https://github.com/Rohman82/portfolio.git
cd portfolio
npm install
npm run dev
```
